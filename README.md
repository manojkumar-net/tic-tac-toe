# Tic Tac Toe - Game 2 Players Only Console Level.

[Wikipedia Reference](https://en.wikipedia.org/wiki/Tic-tac-toe)

## Installation

```sh
git clone https://github.com/manojkumarlinux/tic-tac-toe.git
cd tic-tac-toe
g++ -o game game.cpp
./game
```
